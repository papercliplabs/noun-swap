# NounSwap Web App

NounSwap web app built using [Nextjs](https://nextjs.org)

## Local Development

Copy `.env.example` to `.env`

```bash
cp .env.example .env
```

Populate the `.env` file

Install dependencies

```bash
yarn
```

Start dev server

```bash
yarn dev
```

Build for production (before pushing)

```bash
yarn build
```
