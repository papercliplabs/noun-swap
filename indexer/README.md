# NounSwap Indexer App

NounSwap indexer app built using [ponder](https://ponder.sh/)

## Local Development

Copy `.env.example` to `.env`

```bash
cp .env.example .env
```

Populate the `.env` file

Install dependencies

```bash
pnpm
```

Start dev server

```bash
pnpm dev
```

Build for production (before pushing)

```bash
pnpm build
```
